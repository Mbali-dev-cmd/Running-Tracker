# 🏃🏾‍♀️ My Running Tracker

A multi-language running tracker app built with C++, Python, and HTML/CSS/JavaScript.

## Features
- Log runs with distance and time
- Calculate pace automatically
- Track total distance, average pace and longest run
- Progress graph showing distance over time
- Run history table
- Motivational messages

## Built With
- C++ — Pace calculator module
- Python — Run logging and statistics calculations
- HTML/CSS/JavaScript — Interactive dashboard with Chart.js

## How to Use
1. Run 'Run_logger.py' to log a new run
2. Run 'Calculations.py' to update your stats
3. Open 'index.html' in your browser to view your dashboard
